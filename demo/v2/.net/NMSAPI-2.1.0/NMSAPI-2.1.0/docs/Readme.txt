************************************************中文*****************************************************
发布介绍:

本客户端旨在为使用百度网盟推广API的程序员提供简单的向导和范例。客户端内的示例代码并不完全涵盖百度网盟推广API。
随着百度网盟推广API开放接口的快速增加，本客户端不一定能及时更新。
请参考http://apihome.baidu.com上的开发文档寻找最新的接口和资料。

使用过程中如果遇到任何问题，请联系apihelp@baidu.com并在标题注明《BAIDU NMSAPI V2 .Net Demo使用过程问题求助》

注意事项:

1. 如何设置SOAP Header ?
	参见HeaderUtil.cs。您需要填写username、password和token。请您填写这些信息后再尝试使用此客户端。

2. 如何处理SOAP Response Header 数据 ?
	ResHeader resHeader = client.getAccountInfo(authHeader, ref account);
	返回值就是ResHeader。参见v2中AccountServiceExamples.cs。

3. 如何访问沙盒环境
	请参见SandboxAccountServiceExamples.cs。

4. 如何设置原始类型的参数？
	.Net Framework自动生成的代理类中，针对原始类型，会产生两个属性。
	比如GroupServiceExamples.cs中的GroupType的campaignId属性，
	需要同时设置campaignId和campaignIdSpecified。
	大部分原始类型都会有这两个字段：
	xxx和xxxSpecified。需要设置xxxSpecified=true才能把xxx传送到服务器上。对于没有xxxSpecified的字段则不用设置。
	
5. 本客户端的运行环境
	本示例代码在Visual Studio 2010 RTM版，Microsoft .Net Framework 4.0 下测试通过。

Release Notes:

This client demo is for .Net programer without Web Service Developing experience.
This client is only for demostration, not every function or parameter has been covered. Please see 
online document for details:
http://apihome.baidu.com/docs/index.php/Cn_nms_v2_sp_introduction_aboutThis/

Please contact apihelp@baidu.com with email title: "Baidu NMSAPI V2 .Net Demo Help" if any questions.

KeyPoints:

1. How to set the SOAP Header?
	See HeaderUtil.cs. You will need to fill in your username, password & token before using this client.

2. How to deal with SOAP Response Header data?
	Please see AccountServiceExamples.cs under folder v2
	ResHeader resHeader = client.getAccountInfo(authHeader, ref account);
	The return value is just the SOAP response Header.

3. How to access SandBox?
	See SandboxAccountServiceExamples.cs.

4. How to set attributes of primitive type?
	There are some primitive type attributes in the stub classes generated by .Net Framework.
	If you want to set them and send the data to Baidu API server, you need to set two fields: xxx and xxxSpecified
	For example, in GroupServiceExamples.cs, I want to set campaignId for GroupType,
	I need to set both campaignId and campaignIdSpecified.
	You will need to just do the same. If xxxSpecified not found, just ignore this part.

5. Running environment
	This client is tested under Visual Studio 2010 RTM Version, Microsoft .Net Framework 4.0 Run Time Environment.
